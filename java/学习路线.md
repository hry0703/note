# Java 后端开发学习路线

## 📚 学习路线概览

本学习路线分为三个阶段：**初级阶段** → **中级阶段** → **高级阶段**

---

## 🌱 初级阶段（0-1年）

### 1. Java 语言基础
- [x] Java 语法基础：变量、数据类型、运算符、控制流程
- [x] 面向对象编程：封装、继承、多态
- [x] 集合框架：List、Set、Map 等常用集合
- [x] 异常处理：try-catch、自定义异常
- [x] 文件 I/O：文件读写操作

### 2. Java 进阶特性
- [x] 泛型：泛型类、泛型方法、通配符
- [x] 反射机制：Class 类、动态加载
- [ ] 注解：自定义注解、元注解
- [ ] Java 8+ 特性：
  - [ ] Lambda 表达式
  - [ ] Stream API
  - [ ] Optional 类
  - [ ] 函数式接口

### 3. Java 并发编程
- [ ] 多线程基础：Thread、Runnable
- [ ] 线程同步：synchronized、volatile
- [ ] 线程池：ExecutorService、ThreadPoolExecutor
- [ ] 并发集合：ConcurrentHashMap、BlockingQueue
- [ ] JUC 包：Lock、CountDownLatch、Semaphore 等

### 4. 数据库技术
- [ ] SQL 基础：增删改查、多表查询
- [ ] MySQL/PostgreSQL：安装、配置、基本操作
- [ ] 数据库设计：表设计、索引、外键
- [ ] 数据库优化：索引优化、SQL 优化
- [ ] 事务：ACID 特性、事务隔离级别

### 5. ORM 框架
- [ ] MyBatis：配置、Mapper、动态 SQL
- [ ] JPA/Hibernate：实体映射、关系映射
- [ ] 数据库连接池：HikariCP、Druid
- [ ] 分页查询：PageHelper、JPA 分页

### 6. Spring Boot 框架
- [ ] Spring Boot 基础：自动配置、Starter
- [ ] Web 开发：Controller、RequestMapping
- [ ] RESTful API 设计
- [ ] 配置文件：application.yml、application.properties
- [ ] 依赖注入：@Autowired、@Component

### 7. Spring 核心框架
- [ ] Spring IoC：控制反转、依赖注入
- [ ] Spring AOP：面向切面编程
- [ ] Spring MVC：请求处理流程
- [ ] 事务管理：@Transactional
- [ ] Spring Security：认证与授权

### 8. 缓存技术
- [ ] Redis 基础：安装、数据类型
- [ ] Redis 命令：String、Hash、List、Set、ZSet
- [ ] Redis 持久化：RDB、AOF
- [ ] 缓存策略：缓存穿透、缓存雪崩、缓存击穿
- [ ] Spring Cache：@Cacheable、@CacheEvict

### 9. 项目实战（第一阶段）
- [ ] 使用 Spring Boot + MySQL + MyBatis 开发项目
- [ ] 集成 Redis 缓存
- [ ] 实现用户认证功能
- [ ] 编写单元测试和集成测试
- [ ] 项目部署上线

---

## 🚀 中级阶段（1-3年）

### 10. Spring Cloud 微服务
- [ ] 微服务架构概念：服务拆分、服务治理
- [ ] Eureka：服务注册与发现
- [ ] Spring Cloud Gateway：API 网关
- [ ] Spring Cloud Config：配置中心
- [ ] OpenFeign：服务间调用
- [ ] Hystrix/Sentinel：服务熔断与降级

### 11. 消息队列
- [ ] 消息队列基础概念：生产者、消费者、队列
- [ ] RabbitMQ：
  - [ ] 安装与配置
  - [ ] 交换机类型：Direct、Topic、Fanout
  - [ ] 消息确认机制
- [ ] Kafka：
  - [ ] 基础概念：Topic、Partition、Consumer Group
  - [ ] 高吞吐量场景应用
- [ ] 消息可靠性：消息持久化、幂等性

### 12. 分布式系统
- [ ] CAP 理论：一致性、可用性、分区容错性
- [ ] BASE 理论：最终一致性
- [ ] 分布式事务：
  - [ ] 两阶段提交（2PC）
  - [ ] TCC 模式
  - [ ] Seata 框架
- [ ] 服务治理：限流、熔断、降级
- [ ] 分布式锁：Redis 分布式锁、Zookeeper

### 13. 搜索引擎
- [ ] Elasticsearch 基础：安装、基本概念
- [ ] 索引操作：创建、删除、更新
- [ ] 查询 DSL：match、term、bool 查询
- [ ] 聚合查询：统计、分组
- [ ] 索引优化：分片、副本、映射优化
- [ ] Spring Data Elasticsearch 集成

### 14. 项目实战（第二阶段）
- [ ] 微服务架构项目：拆分多个服务
- [ ] 集成消息队列处理异步任务
- [ ] 实现分布式事务
- [ ] 使用 Elasticsearch 实现搜索功能
- [ ] 服务监控与日志收集

  - [ ] 缓存优化
- [ ] 压力测试：JMeter、压测工具使用

### 19. 设计模式
- [ ] 创建型模式：单例、工厂、建造者
- [ ] 结构型模式：适配器、装饰器、代理
- [ ] 行为型模式：观察者、策略、模板方法
- [ ] Spring 框架中的设计模式应用
- [ ] 实际项目中的模式应用

### 20. 数据结构与算法
- [ ] 数据结构：
  - [ ] 数组、链表、栈、队列

- [ ] 算法：

  - [ ] 动态规划、贪心算法

## 💡 学习建议

1. **循序渐进**：按照阶段顺序学习，不要跳跃
2. **理论结合实践**：每学完一个知识点，立即动手实践
3. **项目驱动**：通过实际项目巩固所学知识
4. **持续学习**：技术更新快，保持学习习惯
5. **总结归纳**：定期总结学习笔记，形成知识体系
6. **参与社区**：加入技术社区，与他人交流学习
