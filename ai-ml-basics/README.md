## 快速导航

### 核心概念
- **Agent**：能够自主决策和执行任务的智能体
- **RAG**：结合检索和生成的 AI 系统
- **Transform**：Transformer 架构，现代 NLP 的基础

### 技术栈
- **机器学习**：让计算机从数据中学习
- **深度学习**：使用神经网络进行学习
- **Attention**：注意力机制，理解上下文的关键

### 实践应用
- **模型推理**：使用训练好的模型进行预测
- **模型优化**：提高模型性能和效率
- **Function Calling**：让 AI 调用外部函数

### 协议和框架
- **MCP**：模型上下文协议
- **Skill**：可复用的 AI 能力模块
