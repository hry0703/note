# 智能办公系统 - 产品使用手册

版本：v2.0 | 更新日期：2024年1月

---

## 一、产品概述

智能办公系统是一款面向中小企业的协同办公平台，提供任务管理、日程安排、文档协作、考勤打卡等核心功能。支持 Web 端和移动端多平台使用。

### 核心功能

- **任务管理**：创建、分配、跟踪任务，支持子任务和截止日期
- **日程日历**：个人/团队日程，支持重复事件和提醒
- **文档协作**：在线编辑、评论、版本历史
- **考勤打卡**：GPS 定位打卡、请假审批、考勤统计

---

## 二、快速入门

### 2.1 账号注册与登录

1. 访问 https://office.example.com 或下载移动端 App
2. 使用企业邮箱或手机号注册
3. 首次登录需设置 6 位数字密码
4. 支持微信、钉钉扫码快捷登录

### 2.2 创建第一个任务

1. 点击左侧「任务」菜单
2. 点击「新建任务」按钮
3. 填写任务标题、描述、负责人、截止日期
4. 可添加子任务，支持拖拽排序
5. 任务状态：待办 → 进行中 → 已完成

## 三、功能详解

### 3.1 任务管理

**创建任务**
- 必填项：标题、负责人
- 可选项：描述、截止日期、优先级（高/中/低）、标签

**任务分配**
- 可将任务分配给团队成员
- 支持多人协作，每人可设置自己的完成进度
- 任务变更会推送通知给相关人员

**任务筛选**
- 按状态：待办/进行中/已完成
- 按负责人：我创建的/分配给我的/我参与的
- 按时间：今日/本周/逾期/无截止日期

### 3.2 日程管理

**添加日程**
- 支持全天事件和定时事件
- 可设置提前 15 分钟/1 小时/1 天提醒
- 支持重复：每天/每周/每月/每年

**日历视图**
- 日视图、周视图、月视图切换
- 支持拖拽调整日程时间
- 可同步到系统日历（需授权）

### 3.3 文档协作

**创建文档**
- 支持富文本、Markdown、表格
- 实时多人编辑，自动保存
- 支持 @ 提及成员

**权限设置**
- 仅自己可见
- 指定成员可编辑/可查看
- 全公司可见

**版本历史**
- 自动保存每次修改
- 可查看、对比、恢复历史版本
- 保留 30 天内的版本记录

### 3.4 考勤打卡

**打卡规则**
- 上班打卡：9:00 前有效
- 下班打卡：18:00 后有效
- 需开启 GPS，允许定位偏差 500 米内

**请假流程**
1. 提交请假申请（事假/病假/年假）
2. 直属上级审批
3. 审批通过后自动同步考勤

**考勤统计**
- 每月 1 号生成上月考勤报表
- 可导出 Excel 用于薪资核算
- 支持补卡申请（需审批）

## 四、常见问题

**Q：忘记密码怎么办？**
通过注册邮箱或手机号重置，或联系管理员重置。

**Q：移动端和 Web 端数据同步吗？**
是的，实时同步，多端登录同一账号即可。

**Q：文档可以导出吗？**
支持导出为 Word、PDF 格式，在文档右上角「更多」中选择。

**Q：考勤打卡失败怎么办？**
检查网络和 GPS 权限，可提交补卡申请由管理员审批。
