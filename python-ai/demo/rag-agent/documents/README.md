本目录包含用于 RAG（检索增强生成）文档问答 Agent 的示例文档。

| 文件 | 内容 | 可测试的问题示例 |

## 使用方式

在开发文档问答 Agent 时，将这些文档作为知识库进行向量化存储，用户提问时检索相关片段并生成回答。
